# AWS SBG REC — Next.js + TypeScript Homepage

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Plus Jakarta Sans** font
- Pure CSS-in-JS (no Tailwind needed)

## How to run

### Step 1 — Open folder in VS Code
File → Open Folder → select `aws-sbg-rec`

### Step 2 — Open terminal (Ctrl + `)

### Step 3 — Install dependencies
```
npm install
```

### Step 4 — Run dev server
```
npm run dev
```

### Step 5 — Open browser
Go to: http://localhost:3000

## Sections
1. **Navbar** — sticky glass navbar with gradient logo
2. **Hero** — headline + JourneyCard (interactive beginner→expert progression)
3. **Stats** — animated count-up stat cards
4. **Features** — 8-feature grid with hover glow
5. **Events** — event cards inspired by reference image (date + type badge + RSVP)
6. **Domains** — infinite marquee of cloud disciplines
7. **CTA** — dark section with gradient headline
8. **Footer** — dark footer with socials
